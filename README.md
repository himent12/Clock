# Clock Suite

Modern desktop clock app built with **Python + PySide6**.

## Quick start

### Windows
1. Double-click `run.bat`.

(First run installs everything automatically.)

### Linux/macOS
```bash
./run.sh
```

## What you get
- World Clock (24h, seconds, UTC, dark mode, always-on-top)
- Stopwatch
- Timer
- Shortcuts: `Space`, `Ctrl+C`, `Ctrl+Q`

## Helpful files
- `run.bat` / `run.sh` → start app
- `install.bat` → install dependencies manually
- `remove.bat` → clean local dev artifacts
